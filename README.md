# Por quê?
Este projeto faz parte do meu portifólio,encontrei ele na internet e resolvi faze-lo.

# Observações
No desafio é necessário consultar uma API e com os dados que ela retorna monstar uma tabela e o gráfico de pizza. Devido a este ser um desafio antigo a API não está mais online.Assim sendo, simulei as chamadas a API com um useEffect que alterava o estado do componente com dados estáticos.

# Iniciando

Clone o repositório

  `git clone https://github.com/lucasluz99/cubo-challenge.git`

Entre na pasta do projeto

`cd cubo-challenge`

Instale as dependêcias

`yarn`

Rode a aplicação

`yarn start`
